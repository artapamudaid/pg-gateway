Cara Penggunaan

Pengujian (Hit Callback)

Kirimkan request POST ke URL gateway ini untuk diteruskan secara otomatis ke Target URL Anda sesuai dengan Environment dan Routing Code yang terdaftar.

1. Format URL Gateway

  • Production: /flip/callback atau /midtrans/callback
  • Sandbox: /sandbox/flip/callback atau /sandbox/midtrans/callback

2. Contoh Request CURL (Flip)

curl -X POST http://127.0.0.1:3131/sandbox/flip/callback \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "token=ISI_DENGAN_PROVIDER_TOKEN_ANDA" \
  -d 'data={"reference_id": "KODE_ROUTING_ANDA-INV-001", "status": "SUCCESS", "amount": 50000}'

3. Contoh Request CURL (Midtrans)

curl -X POST http://127.0.0.1:3131/sandbox/midtrans/callback \
  -H "Content-Type: application/json" \
  -H "Authorization: ISI_DENGAN_PROVIDER_TOKEN_ANDA" \
  -d '{
    "order_id": "KODE_ROUTING_ANDA-ORD-002",
    "transaction_status": "settlement",
    "gross_amount": "100000.00"
  }'

⚠️ Catatan Penting: Parameter reference_id (Flip) atau order_id (Midtrans) WAJIB diawali dengan Routing Code aplikasi yang terdaftar (contoh: SHOP-123 jika routing code adalah SHOP). Jika tidak, gateway tidak akan tahu kemana data harus diteruskan.
